# ⚡ Tunisian Media NLP Assistant (Browser Extension)

إضافة ذكية، حرة، وديناميكية بالكامل لمتصفحات كروم (Google Chrome Extension) مصممة للتحكم التفاعلي في مشغلات الفيديو على منصتي يوتيوب وتيكتوك بناءً على إدخال ومفاهيم المستخدم يدوياً في الوقت الفعلي.

## 🎯 فكرة عمل الأداة ومميزاتها (Core Features)
على عكس الأدوات الصلبة ذات القواميس المغلقة، تم بناء هذه الإضافة لتوفر حرية كاملة للمستخدم في إدارة وضبط سياقات تصفح المحتوى المرئي؛ حيث تتيح الواجهة التفاعلية للأداة:
1. **التحديد اليدوي (Custom Input)**: يقوم المستخدم بكتابة أي كلمة مفتاحية يود مراقبتها، مع تدوين المعنى أو الاستفسار المرتبط بها.
2. **الاستماع الفوري (Live Audio Monitoring)**: تشغيل واجهة برمجة تطبيقات المتصفح (`Web Speech API`) للإنصات الصوتي لـ "الدارجة التونسية واللسان العام" المسموع في الفيديو.
3. **الإيقاف التلقائي (Auto-Pause)**: بمجرد نطق الفيديو للكلمة المحددة، يُرسل المحرك إشارة ميكانيكية فورية للمشغل لإيقاف العرض (`video.pause()`).
4. **الواجهة الجانبية (UI Sidebar Panel)**: حقن نافذة جانبية منسقة تلقائياً على الشاشة تعرض المصطلح المسموع والمعنى المكتوب يدوياً مع أزرار لمواصلة الفيديو أو طلب استفسار.

---

## 📂 بنية المستودع (Repository Structure)
* `manifest.json`: الملف الإداري لتحديد هوية الإضافة، رقم الإصدار، وصلاحيات الحقن والتحكم في نطاقات `youtube.com` و`tiktok.com`.
* `content.js`: المحرك التنفيذي المستقل (جافاسكريبت) المسؤول عن بناء واجهة الإدخال، إدارة الميكروفون، والمطابقة البرمجية الصوتية.

---

## 🛠️ كيفية تنصيب وتشغيل الأداة محلياً (Installation Guide)
بصفتها أداة مفتوحة المصدر، يمكنك تجربة الإضافة وتفعيلها في حاسوبك الشخصي عبر الخطوات التالية:

1. **تحميل الكود**: اضغط على زر **Code** الأخضر في الأعلى، ثم اختر **Download ZIP** وفك ضغط المجلد في حاسوبك.
2. **فتح صفحة الإضافات**: افتح متصفح Google Chrome واذهب للرابط التالي: `chrome://extensions/`.
3. **تفعيل وضع المطور**: قم بتفعيل خيار **Developer mode** (وضع المطور) في أعلى يمين الصفحة.
4. **تحميل المجلد**: اضغط على زر **Load unpacked** الموجود في أقصى اليسار، واهبط بالمؤشر لتحديد مجلد المشروع الذي فككت ضغطه.
